# LLZO-Synthetic: SHAP Pipeline Validation on Synthetic LLZO Data

Random Forest + SHAP pipeline for decomposing processing–composition 

effects on Li-ion conductivity in LLZO solid electrolytes (synthetic 

testbed). Designed as a methodological precursor to "Lithium-Solid-State-Electrolyte-SHAP", which applies the same framework to real experimental data.



## Background

Before applying SHAP-based interpretation to real experimental garnet 
data (where sample size and feature coverage are limited), we validate 
the pipeline on a synthetic LLZO dataset designed to encode known 
physical trends — sintering-driven densification, dopant substitution 
saturation, and Li-site compensation. If the pipeline cannot recover 
these encoded structure–processing–property relationships, it cannot 
be trusted on real data.

**Successor project**: [Lithium-Solid-State-Electrolyte-SHAP](https://github.com/DooHyun2/Lithium-Solid-State-Electrolyte-SHAP) — 
the same framework applied to experimentally measured ionic conductivities.



## Synthetic Dataset

Generated by `data_synth_LLZO.py` (N = 400 samples)

| Feature       | Range          | Encoded behavior |
|---------------|----------------|------------------|
| `sinter_temp` | 900–1250 °C    | Quadratic with optimum near 1160 °C (densification → over-sintering) |
| `dopant_frac` | 0.00–0.25      | Quadratic peaking near 0.27 (saturation) |
| `Li_excess`   | 0.00–0.20      | Quadratic peaking near 0.24 (volatilization compensation) |
| `dopant_code` | {0–4}: Al, Ga, Nb, Ta, Y | Categorical offset; Ta highest, Al baseline |
| `grain_size`  | 1–20 µm        | Weak log-monotonic contribution |
| `sigma_ion`   | normalized score | Target (linear; see note below) |

Note on target transformation : This synthetic testbed uses a linear 
target due to its bounded positive range. The successor project 
(Lithium-Solid-State-Electrolyte-SHAP) applies log10 transformation on real experimental data 
because actual ionic conductivity spans multiple orders of magnitude 
(10⁻⁶–10⁻³ S/cm). This deliberate difference reflects appropriate 
target choice for each data type.

## Method

- Random Forest Regressor ('n_estimators=400', 'random_state=43')
- 25% test split + 5-fold cross-validation for stable evaluation
- SHAP 'TreeExplainer' computed over the full N=400 dataset
- Top-3 features visualized with 'dependence_plot'

## Model Performance

| Metric                  | Value             |
|-------------------------|-------------------|
| Test R² (25% split)     | 0.882             |
| Test MAE                | 0.062             |
| 5-fold CV R² (mean ± std) | 0.895 ± 0.021 |

The narrow CV std (±0.021) and near-equality between Test and CV R² 
confirm stable generalization with no overfitting. The high R² reflects 
controlled noise (σ = 0.05) and complete feature coverage in the 
synthetic data — by contrast, Lithium-Solid-State-Electrolyte-SHAP on real experimental data 
achieves LOOCV R² ≈ 0.698, illustrating the gap between controlled 
testbeds and experimental settings.

The sintering curve shows a clear **densification → optimum → 
post-peak decline** structure: SHAP values rise sharply from 900 °C, 
plateau around 1100–1160 °C, then decline slightly above 1200 °C. 
This non-monotonic shape reproduces the well-known LLZO sintering 
trade-off — densification at moderate temperature followed by Li 
volatilization losses at higher temperature. Color (dopant_frac) is 
well-mixed vertically, indicating the sintering effect is largely 
independent of dopant loading in this range.

## Summary

The RF + SHAP pipeline recovers **three physically meaningful trends** 
from the synthetic LLZO data:

1. Sintering-driven densification with a clear optimum near 
   1100–1160 °C — both the rise and the post-peak decline are 
   recovered, validating that SHAP captures non-monotonic dependencies, 
   not just linear trends.
2. A saturating benefit from dopant substitution, with SHAP 
   contribution plateauing near the encoded optimum.
3. Li excess acting as an independent compositional compensation, 
   with diminishing returns past ~0.15.

These results confirm that the framework correctly extracts encoded 
structure–processing–property relationships, making it suitable for 
transfer to experimental datasets where the true trends are unknown.

Caveat. Because trends are partially encoded into the data 
generator, this analysis primarily validates the SHAP framework's 
ability to recover embedded relationships rather than discover new 
physics. Discovery on real data is the role of Lithium-Solid-State-Electrolyte-SHAP.
